Create a "model break view" from a model. This uses a "ModelBreakManager" sample and csg.js to perform boolean operations on the tessellated Communicator model.

  1. Select an axis to create a break view along using the drop down.
  2. Position the "break" planes at the desired cut locations (the section of the model between the break planes will be hidden).
  3. Click "break" to create a break view.
  4. The break view ID is added to the web page. Click the "Reset" button to restore the model or a view ID to restore a specific break view.
